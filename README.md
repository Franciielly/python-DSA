# 🐍Python: Do Básico à Aplicação em IA

Repositório com exercícios, projetos e anotações do curso Python Fundamentos para Análise de Dados e Data Science da Data Science Academy.

O objetivo deste repositório é documentar minha evolução no aprendizado de Python aplicado à análise de dados, automação e inteligência artificial, passando desde conceitos básicos da linguagem até aplicações práticas em Data Science e IA.
